YouTube Downloader API Documentation

Welcome to the YouTube Downloader API! Below are the available endpoints to download audio and video from YouTube.

Download Audio

Endpoint: /download/audio

Query Parameters:

  • url (required): The URL of the YouTube video

Example: /download/audio?url=https://www.youtube.com/watch?v=YOUR_VIDEO_ID

Download Video

Endpoint: /download/video

Query Parameters:

  • url (required): The URL of the YouTube video

Example: /download/video?url=https://www.youtube.com/watch?v=YOUR_VIDEO_ID
