API Documentation

Download Video

Endpoint: /download

Method: GET

Parameters:

  • url: YouTube video URL (required)
  • format: Desired format (e.g., mp4, audio, 360p, 480p, 720p, 1080p, 1440p, 2160p) (optional, default: mp4)

Get Video Metadata

Endpoint: /metadata

Method: GET

Parameters:

  • url: YouTube video URL (required)

Search Videos

Endpoint: /search

Method: GET

Parameters:

  • query: Search query (required)
